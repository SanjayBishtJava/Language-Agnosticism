## Installing Erlang
https://brewinstall.org/install-erlang-on-mac-with-brew/

## Compiling and Executing
- cd helloworld
- `erlc *.erl` will compile 

- `erl -noshell -s helloworld start -s init stop` will execute and prints output on console.
